NeoVerse AI Dashboard
A futuristic, responsive AI-powered dashboard built using HTML5, CSS3, Flexbox, CSS Grid,and JavaScript. This project showcases 
modern UI/UX principles, advanced animations, and mobile-first responsive design, making it an ideal Full Stack Development (FSD)
internship project.

Project Overview

NeoVerse AI Dashboard is a next-generation technology platform interface that demonstrates modern frontend development concepts. 
The dashboard features interactive analytics, AI service cards, futuristic animations, glassmorphism effects, and an AI 
Communication Terminal.

The project is designed to help interns master:

* Responsive Web Design
* CSS Flexbox
* CSS Grid
* Mobile-First Development
* Modern UI/UX Design
* JavaScript DOM Manipulation
* CSS Animations & Effects
